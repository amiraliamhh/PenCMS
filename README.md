## Pen CMS
<h4>PenCMS, is a lightweight CMS developed for bloggers.</h4>
<ul>
    <li>You can just download the source code, no need for `npm install`.</li>
    <li>You should have MongoDB installed and running.</li>
    <li>You can run `npm start` to start the server on http://127.0.0.1:4567</li>
</ul>

## Aboout
PenCMS is developed to be used alongside other frameworks. But, you can also use PenCMS without any other framework if you prefer. 
It's main goal is to be used by bloggers, content creators, digital marketers, marketing teams, etc.
